# ICSE Percentage Calculator

A clean, distraction-free percentage calculator built specifically for ICSE students.

## Why this exists
ICSE students are often required to manually calculate percentages using calculators.
This app removes that friction with a simple, accurate, and student-friendly interface.

## Features
- Professional UI
- Instant percentage calculation
- Input validation
- Mobile-friendly design
- No login, no ads, no nonsense
